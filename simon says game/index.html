ℹ️ About Simon Says

This is a modern web implementation of the classic electronic memory game, **Simon**. The goal is simple: repeat the sequence of flashing colors and sounds exactly as the game presents them.

Each successful round adds another step to the sequence, testing your short-term auditory and visual memory to the limit.

Built using **HTML, CSS, and JavaScript** for a fun, responsive experience!

🧠 How to Play

Ready to challenge your memory? Follow these steps:

  • **Start:** Select a mode and click **"Start Game"** or press a key.
  • **Watch:** The game will flash one of the four color buttons and play a corresponding sound.
  • **Repeat:** Click the same color button that just flashed.
  • **Advance:** If correct, the game will flash the original pattern *plus* one new color.
  • **Goal:** Continue repeating the increasingly longer sequence until you make a mistake.
  • **Game Over:** If you click the wrong button, the game ends, and your score will be added to the Leaderboard!
